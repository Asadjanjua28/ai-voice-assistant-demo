## 🔧 Requirements

Install dependencies using pip:

```bash
pip install -r requirements.txt

Or manually install:pip install speechrecognition pyttsx3 wikipedia requests


## ▶️ How to Run
Make sure your microphone is working and run the assistant using: python voice_assistant_demo.py

Or full path example:python D:\PROJECTS\voice_assistant_demo.py

🛠️ Optional: Weather Feature
To enable the weather feature, get a free API key from https://www.weatherapi.com and replace:

get_weather(city, api_key="your_api_key_here")
With your actual API key in the script.
